# 자료구조, 알고리즘 공부
자료구조, 알고리즘 강의를 들으며 정리해보는 레포

### 자료구조 (도구)
: 메모리를 효율적으로 사용하며 빠르고 안정적으로 데이터를 처리하는 것이 목표
- 선형 구조: 배열, 연결 리스트, 스택, 큐
- 비선형 구조: 트리, 그래프


### 알고리즘 (레시피)
: 특정 문제를 효율적이고 빠르게 해결하는 것이 목표
- Binary Searh 
- Shortest Path
